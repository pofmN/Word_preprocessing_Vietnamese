## init env
`python -m venv venv`
## 1. # activate env
`.\venv\Scripts\activate`
## 2.  upgrade pip
`python -m pip install --upgrade pip`
## 3.  check dependency packages
`pip freeze`
## 4. install dependency packages in file requirements.txt
`pip install -r requirements.txt`


## 5. Run server 
`cd main`
`uvicorn api:app --reload `
